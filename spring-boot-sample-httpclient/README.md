## Spring Boot 集成

### 配置 `pom`

```xml
<properties>
    <httpclient.version>4.5.13</httpclient.version>
</properties>
...
<!-- HttpClient -->
<dependency>
    <groupId>org.apache.httpcomponents</groupId>
    <artifactId>httpclient</artifactId>
    <version>${httpclient.version}</version>
</dependency>
```
